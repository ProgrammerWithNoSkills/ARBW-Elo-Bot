# Simple Ranked Bedwars Elo Bot
## Designed for usability and reliablility

This bot was designed to be an open-source ranked bedwars elo bot that was reliable and simple to use.

## Features

- Auto Elo scoring
- Strong permission control
- Automatic deploy
- Multiserver support
- Open-source
